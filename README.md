# Chrome DevTools Plugin

## Project Overview

The Chrome DevTools plugin provides developers with tools to enhance their web debugging and performance analysis capabilities. This project aims to create a seamless integration with Chrome's built-in developer tools, allowing for efficient workflows and improved productivity.

## Features

- **Performance Monitoring**: Track the performance of web applications in real-time.
- **Customizable UI**: Fully customizable user interface to suit various development needs.
- **Code Snippet Management**: Save and manage commonly used code snippets directly within DevTools.
- **Error Logging**: Simplify error logging with a user-friendly interface.

## Installation

To install the Chrome DevTools plugin, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/Jamie78herndon/my-brew.git
   ```
2. Navigate to the project directory:
   ```bash
   cd my-brew
   ```
3. Open Chrome and navigate to `chrome://extensions`.
4. Enable "Developer mode" in the top right corner.
5. Click on "Load unpacked" and select the project directory.

## Usage

After installation, open Chrome DevTools by right-clicking on a web page and selecting "Inspect". You will find the plugin integrated into the DevTools panel.
